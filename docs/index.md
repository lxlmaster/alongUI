# alongUI

alongUI 是一个面向 Vue 3 的 PC 中后台组件库，目标是在完整组件覆盖、清晰 API 与克制视觉之间取得平衡。

当前仓库处于工程骨架阶段，已完成 Button、Input、Icon、Tooltip、Loading 的基础实现与文档模板。
