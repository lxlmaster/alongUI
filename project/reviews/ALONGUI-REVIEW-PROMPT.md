# alongUI · Reviewer Prompt

> iOS 精确设计语言审查提示词

---

## 审查红线（一条不过就退回升）

```
1. 按钮是不是文字按钮？（如果是填充按钮，圆角是不是 12px？）
2. 输入框是不是灰底 #F2F2F7、无边框、10pt 圆角？
3. Switch 开启态是不是 #34C759（iOS 绿）？
4. 所有可交互元素的颜色是不是 #007AFF（不是 #1D1D1F）？
5. 焦点样式有没有用黑色 outline？（iOS 没有黑色 outline）
6. 占位符文字是不是 #8E8E93？
7. 禁用态是不是只用 opacity:0.4？
8. 红色是 #FF3B30 吗？
```
